Registro de socio Complétalo para registrarte como socio aliado, así accederás a las credenciales de los productos que tenemos para ti. ¿Cuál es el proceso? 1. Proporciona tus datos Deberás de comunicarte con el equipo de soporte de Interbank para entregarle los siguientes datos del responsable técnico:
  • Nombre completo
  • Correo electrónico corporativo
2. Recibe tus credenciales de prueba Una vez registrado te serán entregados los siguientes datos:
  • Credenciales de prueba: Tu id de aplicación y contraseña.
  • Subscription Key: Llave de suscripción que deberás usar como propiedad si se integra a través del Web Component o tambien como cabecera en todas las peticiones de integrarse mediante el API de Interbank.
Recuerda que estas credenciales son para las pruebas funcionales en un ambiente controlado, no se deben de usar para salir a producción. 3. Genera credenciales de producción Al final de la integración, te crearemos otras credenciales especiales cuando:
  • Cumplas todo el paso a paso de integración del producto que quieras integrar
  • Las pruebas funcionales tengan el visto bueno de Interbank
  • El contrato comercial este firmado
