Configuración de estilos ¡Personalizar cada Web Component de Interbank es posible! Puedes configurar la paleta de colores o cambiar su comportamiento para que se adapte de la mejor manera a tu página web.
Personalizar paleta de colores Cada Web Component sigue un diseño al estilo de Material Design por lo que para configurar los colores que serán usados dentro de todo el componente se pueden definir tres colores ( primary, secondary y accent) que deberán ser indicados en atributos con notación HEXADECIMAL:
Personalizar tipografía Cambiar la tipografía del componente es posible según la identidad de tu página. Primero las tipografías a usar deberán ser cargadas como se hace normalmente en el documento: Se cargarán un total de 3 fuentes, esto debido a que el web component usa tres tipos: Regular, Semi-bold y Bold.
