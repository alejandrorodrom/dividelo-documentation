Integrar componente en Ficha de Producto (Previo al pago)
Mediante este componente podrás incluir en tu ficha de producto una simulación de cuotas para clientes exclusivos de Interbank. El script que carga Divídelo se encuentra desplegado en un CDN. Es suficiente con importar el script dentro de tu página web y usar el componente como un Custom Element. Al interactuar con el link para conocer más el usuario verá un popup o modal en donde completará su proceso de pago.
Atributos
  1. subscription-key (obligatorio): Credencial generada por Interbank.
  2. amount (obligatorio): Será el monto con el cual se ejecutará una simulación.
  3. logo-url (obligatorio): URL del logo del comercio (aparecerá en el popup de simulación).
  4. currency (opcional): Cadena de caracteres indicando la moneda que se debe usar para la simulación, recibe los valores USD (Dólares) o PEN (Soles). Por defecto, el valor es PEN.
