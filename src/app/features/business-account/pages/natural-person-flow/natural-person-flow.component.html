Flujo - Persona Natural El flujo de apertura de cuenta a través del WebComponent de Interbank puede variar entre 3 a 5 pasos, esto dependerá de si la persona que quiere abrir una cuenta es cliente de Interbank o no y además, posee o no una tarjeta débito. A continuación se describirá en detalle cada paso del proceso además de posibles configuraciones adicionales.
Identificación de cliente Será el primer paso para la apertura de cuenta, en este paso, el cliente deberá proporcionar su DNI para determinar si ya es un cliente Interbank o no. Una vez el cliente haya ingresado su DNI, el sistema determinará si el DNI corresponde a un cliente ya registrado en Interbank, de ser así, el cliente sólo deberá proporcionar información adicional sobre la moneda y otros detalles como impuestos de renta. Si el sistema determina que el DNI ingresado no corresponde a un cliente registrado en Interbank, se agregará un paso adicional donde se detallará información personal del usuario como país de residencia, direcciones, etc.
2. Proceso de Validación de identidad Una vez se haya identificado al cliente a través de su RUC o DNI, se dará paso a la sección de preguntas KBA, en donde el cliente deberá responder una serie de preguntas para validar su identidad, este paso es obligatorio para continuar con el flujo. Si el cliente responde equívocamente las preguntas, un nuevo conjunto de preguntas será proporcionado para intentar nuevamente, para un total de 3 intentos máximos, caso en el que el DNI será bloqueado y no podrá usarse para abrir una cuenta durante las próximas 24 horas.
Una vez que se responden correctamente las preguntas de seguridad se debe pasar por el segundo factor de autenticación que corresponde a validación por biometria (Validación Facial)
Para poder ejecutar correctamente el proceso de validación sera necesario establecer permisos a la camara en navegador, como se presenta en la imagen.
3. Formulario de Identificación del RUC En este paso si ya es cliente y se encuentra registrado en Interbank Interbank solo sera necesario completar algunos campos del faltantes.
Caso contrario de no ser cliente Interbank será necesario completar datos personales.
De igual manera al no ser cliente se deberá completar los campos del domicilio.
En caso de que el cliente seleccione la opción en la que asegura tener residencia fiscal o tributaria fuera del país, se desplegarán 3 campos adicionales en los cuales se especificará el país, número tributario y un checkbox para especificar si este no es requerido en el país seleccionado. En caso de que el cliente tenga otro país o países adicionales los cuales especificar, el botón Añadir otro país creará cuantos campos adicionales se necesiten para agregar la información necesaria.
4. Configuración de la cuenta Este será el paso final, en el cual el cliente escogerá la moneda (Soles o Dólares) de su preferencia para abrir su cuenta Interbank Finalmente, proporcionará su acuerdo con los términos y condiciones además de otras políticas de tratamiento de datos e información personales.
Se presentara los campos para seleccionar la moneda de la cuenta las opciones Soles, Dólares o Ambos esta ultima opción creara dos cuentas silmultaneamente. Se deben marcar los terminos y condiciones para crear la cuenta.
5. Creación de cuenta exitosa En este punto, el cliente ya habrá creado su cuenta de manera exitosa y es aquí donde se notificará al comercio el evento de creación de cuenta.

Correo de notificación de apertura de cuenta Una vez se hayan completado todos los pasos, el cliente podrá crear su cuenta, momento en el que recibirá un correo electrónico detallando información como el número de cuenta, código CCI y otros datos adicionales. El correo en mención también contendrá dos archivos PDF con el detalle más específico sobre la cuenta creada más otro archivo PDF con el Contrato de Depósitos en Cuenta Corriente.
